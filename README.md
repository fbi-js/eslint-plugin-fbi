# eslint-plugin-fbi
ESLint plugin for fbi

## Installation

```bash
npm install eslint-plugin-fbi --save-dev
```


## Configuration

```js
{
  "extends": [
    // basic
    "plugin:fbi/base"

    // for node
    // "plugin:fbi/node"

    // for vue
    // "plugin:fbi/vue"

    // for react
    // "plugin:fbi/react"
  ],
  "rules": {
    // ...
  }
}
```
